# Bhanu ♥ Lola — Long Distance Love (Static)
This is a ready-to-deploy **static** site. No build tools needed.

## Deploy on Vercel (Drag & Drop)
1. Go to https://vercel.com/new (New Project).
2. Look for the **Drag and Drop** area.
3. Drag the whole `love-site` folder (or zip → unzip → folder) onto that area.
4. After ~1 minute, you'll get a live link like `https://something.vercel.app`.

## Edit Text
Open `index.html`, change names or messages, save, and re-deploy.
